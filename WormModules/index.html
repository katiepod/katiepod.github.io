Tool 3: Module Annotation Pages

Each annotation page provides the following information about a module:

  • Conditions (from a list of diverse 716 perturbations) that activate the module
  • Enriched GO terms
  • Enriched tissues
  • Enriched transcription factor binding site motifs (HOMER)
  • Enriched promoter oligonucleotide words (MobyDick)
  • Enriched 3'UTR oligonucleotide words (MobyDick)
  • Constituent genes

    Note about hemi-modules: Certain module annotations are reported per hemi-module. Hemi-modules are sets of genes within a module that are regulated in opposite directions. Module genes regulated in one direction (up or down) are part of one hemi-module and genes regulated in the opposite direction (down or up) are part of the other hemi-module (the 'up' or 'down' direction is assigned arbitrarily for each module). By definition, the “a” hemi-module is larger (i.e. contains more genes) than the “b” hemi-module. Please see the main page for a graphical overview or (if you are still confused) see our paper for a more detailed explanation.
